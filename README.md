## Running locally

this repo was build and tested with Deno 2.1.2, start by installing packages with

```bash
# install packages
deno i
```

You can run solutions with `deno` like this

```bash
deno ./src/day1.solve.ts -R
```

You can start tests with

```bash
deno test -R
```
